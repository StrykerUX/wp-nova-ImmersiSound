== Description ==

Nova ImmersiSound is a professional WordPress plugin that brings your website to life with immersive sound effects. Map sounds to any CSS class or ID selector, create engaging page transitions, and provide your visitors with a unique audio experience.

= Key Features =

* **CSS Sound Mapping** - Attach sounds to any element using class or ID selectors
* **Page Transitions** - Add entry and exit sounds to specific pages
* **User Controls** - Customizable audio control widget with multiple styles
* **Gutenberg Blocks** - Native blocks for easy integration
* **Mobile Support** - Optional mobile device compatibility
* **Privacy Focused** - Full GDPR compliance with explicit consent
* **Accessibility** - Respects prefers-reduced-motion settings

Nova ImmersiSound provides a JavaScript API for developers who want to integrate custom sound functionality into their themes or plugins.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/nova-immersisound` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Complete the quick setup wizard that appears on activation
4. Use the Nova ImmersiSound screen to configure your sound mappings

== Frequently Asked Questions ==

= Can I use my own sound files? =

Yes! Upload any MP3, WAV, OGG, or M4A file through the WordPress Media Library and use them in your sound mappings.

= Does it work on mobile devices? =

Mobile support is optional and can be enabled in the settings. Note that many mobile browsers have restrictions on autoplay audio.

= Is it compatible with page builders? =

Yes, Nova ImmersiSound works with all major page builders including Elementor, Divi, and Gutenberg.

= Will it slow down my site? =

No, sounds are loaded asynchronously and only when needed. The plugin is optimized for performance.

= Can visitors disable the sounds? =

Yes, the plugin includes a customizable control widget that allows visitors to mute sounds or adjust volume.
